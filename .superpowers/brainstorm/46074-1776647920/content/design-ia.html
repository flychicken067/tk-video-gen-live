MVP 页面结构 & 功能清单

基于现有 mvp_hi_fi.html，增强 4 个功能点 + 部署到香港服务器

整体架构

单页应用 — 4 个 Tab 导航
🛒 买家视图
素材库
  • 免费素材 — 直接下载 .mp4
  • 付费素材 — 弹出微信联系
  • 分类过滤：动物/奢品/潮配/商务…
  • 搜索框
🤖 买家视图
AI 生成（核心差异化）
  • 必选：旋转方式（水平/垂直/螺旋/脉冲）
  • 必选：背景效果（纯黑/粒子/光晕/赛博）
  • 关键词输入（可选，中文）
  • 真实调用即梦 API → 返回图像
  • 生成后可下载或加入购物车
🏪 卖家视图
卖家入驻（真实）
  • 填写基本信息（姓名/微信/设计方向）
  • 上传样品（最多 3 张，/api/upload）
  • 设定定价方式（免费 / 定价）
  • 提交 → server 转发微信/飞书通知你
💰 买家/卖家通用
定制委托
  • 描述需求 + 行业 + 尺寸
  • 提交 → 收到微信回调通知
  • MVP 阶段：人工交付

后端 server.js 新增接口

API 端点（在现有基础上增加）
端点 功能 状态
POST /api/generate AI 生成（含旋转/背景参数） 已有，增强参数
POST /api/seller/apply 卖家入驻申请 → 飞书/微信通知 新增
POST /api/seller/upload 样品上传（multer，存 /uploads） 新增
POST /api/custom/order 定制委托提交 → 通知 新增
GET /api/health 健康检查 已有

部署方案（中国大陆可访问）

腾讯云香港 Lighthouse — 无需 ICP 备案
服务器配置
  • 腾讯云 HK 轻量服务器 2C2G
  • ¥24/月，中国大陆延迟 <50ms
  • Node.js 20 + PM2 守护进程
  • Nginx 反向代理 + HTTPS
域名方案
  • 有已备案域名 → 直接用子域名
  • 没有 → IP 直接访问（MVP 阶段够用）
  • 或用 .com 域名（HK 服务器不需要 ICP）

不在这次 MVP 范围内

❌ 真实支付（需 ICP 经营许可）   ❌ 用户注册/登录    ❌ 素材数据库    ❌ 分销系统

以上全是第二阶段，等第一批客户验证付费意愿后再做。
