部署方案对比

核心问题：Vercel 在中国大陆访问不稳定

⚠️ Vercel 的问题

Vercel 的 .vercel.app 域名在中国大陆访问极慢或间歇性超时，不适合给中国客户演示。技术上也有限制：视频生成需要轮询（30-60秒），Vercel 免费版函数超时只有 10 秒。

A

🆓 Railway（推荐，免费起步）

选 Hong Kong 区域，Node.js 直接部署，无函数超时限制，每月 $5 免费额度够跑 MVP。中国大陆访问正常。

优点

  • 免费额度够用
  • HK 节点，中国可访问
  • 支持长连接/轮询
  • 一键 Git 部署

缺点

  • 需要绑定信用卡验证身份
  • 超出免费额度后按量收费
B

💴 腾讯云 HK 轻量（稳定，¥24/月）

最稳定的中国可访问方案，延迟低，完全可控。需要买服务器（¥24/月），国内用户首选。

优点

  • 延迟最低（中国大陆 <30ms）
  • 完全可控，无限制
  • 后续可直接申请 ICP 备案

缺点

  • 需要花钱（¥24/月）
  • 需要自己配置服务器
C

🆓 Fly.io（免费，HK 节点）

可部署到 Hong Kong 节点，免费套餐包含 3 个共享 CPU + 256MB 内存，中国大陆基本可访问。

优点

  • 完全免费起步
  • HK 节点

缺点

  • 免费机器冷启动慢（30秒+）
  • 内存小，文件上传受限

我的建议

先用 Railway HK 免费跑 MVP，验证客户愿意付费后，再花 ¥24/月迁移到腾讯云正式域名。

Railway 绑卡后每月有 $5 免费额度，MVP 阶段流量小完全够用，不会产生费用。
