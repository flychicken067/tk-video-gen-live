问题 1 — 图片来源

A

用户上传自己的产品图

买家拍一张产品照片上传，AI 把它变成旋转3D全息视频。适合电商卖家、品牌方自定义。

B

上传 + 关键词两种都支持

有图上传图，没图就填关键词（如"金色手表"），先文生图再图生视频，两步合一。

问题 2 — 视频生成 API

A

即梦 I2V（你已有 ARK Key）

ByteDance 火山引擎，模型 doubao-seeddance-1-0-lite-i2v，同一个 API Key 直接用，国内速度快。

优点

  • 无需申请新 Key
  • 延迟低

缺点

  • 需确认账号已开通视频模型
B

快手可灵 Kling API

国内最强图生视频，效果更稳定，3D 旋转质量更好，但需要单独申请 Kling API Key。

优点

  • 业界最佳 I2V 效果
  • 专门的 3D motion 模式

缺点

  • 需额外申请 Key
