AI 生成模块 — 实现方式

用户选旋转方式 + 背景效果 + 关键词 → 生成全息素材

A

真实 AI 接口（推荐）

对接即梦/ComfyUI/Replicate 等 API，用户填参数后真实生成图像。需要 API Key 和生成费用（约 ¥0.5-2/张）。

优点

  • 客户体验真实，付费意愿高
  • 核心差异化能被感受到

缺点

  • 需要接入成本
  • 生成时间 10-30 秒
B

参数选择 → 预置素材匹配

用户选择旋转方式 + 背景效果，系统根据参数组合展示已有素材库中的匹配结果，不调用 AI 接口。

优点

  • 零接入成本，速度极快
  • 可控体验

缺点

  • "AI"感偏弱
  • 素材有限，重复率高
C

提交需求 → 人工交付

用户选参数后生成"定制订单"，由设计师或杨磊团队手动生成后上传回复。48小时内交付。

优点

  • 零技术成本启动
  • 质量最可控

缺点

  • 无即时感，体验弱
  • 难以规模化
